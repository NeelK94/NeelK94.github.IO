Building a chess engine

This project was conducted to combine my passion for chess and python coding.

Goals Results Next steps
Create a chess engine that allows both two player and one player chess. Chess engine should be able to look at any given depth to evaluate future positions. For automated play, minmax algorithm with alpha-beta pruning used. The engine works at depth 2, 3, or 4, depending on complexity of the board. Average thinking time < 5s. Caching of pre-evaluated positions. Multi-thread board evaluation. Save and load game option.
